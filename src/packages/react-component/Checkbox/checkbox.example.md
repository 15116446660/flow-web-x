基础用法
```
<div>
  <Checkbox labelPosition='left' label='Sample' />
  <Checkbox label='Sample' />
  <Checkbox label='Sample Checked' checked />
  <Checkbox label='Sample Disabled' disabled />
  <Checkbox label='Sample Checked Disabled' checked disabled />
</div>
```
### Indeterminate
通常用于全选
```
<div>
  <Checkbox label='Indeterminate' indeterminate />
  <Checkbox label='Disabled' indeterminate disabled />
</div>
```
